🚀 *my-6th-assignment-digitools-platform*
A sleek and interactive digital tools marketplace where users can browse premium digital products and manage their purchases with an integrated cart system.

🛠️ *Technologies Used*
*Core:* HTML5, CSS3, JavaScript (ES6+)

*Library:* React.js

*Styling:* Tailwind CSS & DaisyUI

*Icons:* Lucide-React & Font Awesome

*Notifications:* React-Toastify

*Deployment:* Netlify

*Live Link:* https://assignment-6-digitools-platform.netlify.app/


📝 *Description*
Digitools Platform is a modern web application designed for a seamless shopping experience of digital assets. It features a clean, responsive UI that allows users to explore various tools, track their selections via a dynamic navbar badge, and manage their orders through a functional checkout system.


✨ *Key Features*
1. Dynamic Product Selection & Add to Cart
Users can browse a wide range of digital products. With a single click, items are added to the cart, featuring real-time validation to prevent duplicate entries and instant feedback via Toastify notifications.

2. Live Navbar Integration
The project features a smart Navbar Counter. Whenever an item is added or removed from the cart, the badge on the cart icon updates automatically, allowing users to keep track of their selections from anywhere on the page.

3. Comprehensive Cart Management
The dedicated Cart section allows users to:

*Review Items:* View all selected products with their individual prices.

*Remove Items:* Easily delete specific items they no longer wish to purchase.

*Proceed to Checkout:* Calculate the total price automatically using the reduce method and clear the cart to complete the transaction.
